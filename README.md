## 💼 BUSINESS PROBLEM

A multi-region retail firm suffered from a **12% decline in overall profit margins** over a rolling 18-month period. Management lacked operational visibility due to:
1. **Siloed regional reports:** Transaction records were scattered across offline CSV spreadsheets and legacy on-premise databases, causing massive reporting delays.
2. **Unoptimized inventory allocation:** Warehouses held high stock counts of low-performing items, draining available working capital.
3. **Product performance blindspots:** Leadership could not isolate seasonal demand spikes or track regional profit margins by product line.

---

## 📊 DATASET DESCRIPTION

The core dataset contains over **500,000 retail transaction records** spanning a 3-year timeline (2023 - 2025). The data schema is built as a highly optimized **Star Schema Data Model** inside Power BI, consisting of:

* **Fact_Sales:** ~500k transaction records containing transactional timestamps, quantity sold, unit price, and discount rates.
* **Dim_Product:** Master product records mapping individual SKUs to product categories, tiers, and acquisition costs.
* **Dim_Customer:** Demographics, loyalty status, and signup dates for active shoppers.
* **Dim_Geography:** Multi-level region mapping containing cities, states, and sales zones.
* **Dim_Date:** Standard date table supporting Power BI Time Intelligence calculations.

## 🛠️ DATA CLEANING & ETL PROCESS

Before loading the transaction logs, a rigorous Python Pandas and SQL-based ETL pipeline was executed to guarantee data reliability:

```python
# Programmatic Deduplication & Null Handling in Pandas
import pandas as pd

# Load dataset
df = pd.read_csv("data/raw_uncleaned_sales.csv")

# 1. Deduplicate based on Transaction ID
df.drop_duplicates(subset=["TransactionID"], keep="first", inplace=True)

# 2. Impute null Customer IDs (14% entry failure rate) as 'CUST-GUEST'
df["CustomerID"].fillna("CUST-GUEST", inplace=True)

# 3. Clean Unit Price anomalies (resolving negative pricing records)
df["UnitPrice"] = df["UnitPrice"].abs()
```

### **Core Data Cleaning Steps Achieved:**
1. **Deduplication:** Dropped **15,240 duplicate transaction entries** generated by batch transmission errors.
2. **Null Value Treatment:** Resolved **14% missing Customer IDs** (guest transactions) by mapping them to a standard `'CUST-GUEST'` identifier.
3. **Data Type Correction:** Standardized incoherent date timestamps and unified geographic labels across inconsistent regional names (e.g., standardizing `north`, `North`, and `North-Zone` $\rightarrow$ `North`).

## 💻 SQL ANALYSIS

Relational SQL queries were executed to perform complex database transformations and pre-aggregate business KPIs.

### **Example: Analyzing Product Revenue and Profit Contribution**
```sql
WITH ProductSales AS (
    SELECT 
        p.Category,
        p.ProductName,
        SUM(s.Quantity) as TotalQty,
        SUM(s.Quantity * s.UnitPrice) as GrossRevenue,
        SUM(s.Quantity * p.ProductCost) as TotalCost
    FROM Fact_Sales s
    JOIN Dim_Product p ON s.ProductID = p.ProductID
    GROUP BY p.Category, p.ProductName
)
SELECT 
    Category,
    ProductName,
    GrossRevenue,
    (GrossRevenue - TotalCost) as GrossProfit,
    ROUND(((GrossRevenue - TotalCost) / GrossRevenue) * 100, 2) as ProfitMarginPct,
    DENSE_RANK() OVER (PARTITION BY Category ORDER BY (GrossRevenue - TotalCost) DESC) as ProfitRank
FROM ProductSales;
```

## 🧠 KEY KPIs TRACKED

* **Total Revenue:** Gross transactional billing across all channels.
* **Gross Profit Margin (%):** Calculated as `(Revenue - Cost) / Revenue`, assessing overall business margin health.
* **Year-over-Year (YoY) Sales Growth:** Measure of current-year sales velocity compared against the same historical period.
* **Average Order Value (AOV):** Calculated via DAX measure as `Total Revenue / Total Transaction Count`, tracking shopper purchasing depth.

## 📈 KEY INSIGHTS & RECOMMENDATIONS

### **Strategic Insight:**
* **Inventory Cost Drainage:** Discovered that **3 low-margin product categories** represented **22% of total warehouse inventory space** and capital allocation but generated **less than 4% of total profit**.
* **Seasonal Sales Spike:** Identified that the West Region experiences a **38% surge in home appliance sales** during Q3, coinciding with regional promotional cycles, which previously suffered from stockouts.

### **Actionable Business Recommendations:**
1. **Capital Reallocation:** Implement a **15% inventory reduction** on the identified underperforming product lines. Reallocate this freed capital toward top-tier electronics, resulting in an estimated **$35,000 in working capital savings** and a projected **4.5% boost in quarterly profit margins**.
2. **Predictive Inventory Routing:** Increase stock levels of high-margin home appliances in the West Region by **25% prior to the start of Q3** to capture seasonal demand, preventing out-of-stock occurrences and adding an estimated **$18k in seasonal revenue**.
